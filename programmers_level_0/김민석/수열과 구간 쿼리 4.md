## 수열과 구간 쿼리 4

```java
class Solution {
    public int[] solution(int[] arr, int[][] queries) {
        for(int[] x : queries) {
            for(int i = x[0]; i <= x[1]; i++) {
                if(i % x[2] == 0) {
                    arr[i] += 1;
                }
            }
        }
        return arr;
    }
}
```
